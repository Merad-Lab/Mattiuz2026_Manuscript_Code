# Mattiuz et al 2026: Manuscript Code

This repository contains data and code relevant to spatial transcriptomics datasets presented in [Mattiuz R., et al.](https://www.biorxiv.org/content/10.1101/2024.12.27.628014v1).

### Downloading manuscript data

Raw and processed Visium SD and MERFISH data are available on NCBI GEO under accession number: [GSE322553](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE322553) and [GSE327192](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE327192).

## Description of files

This repository contains code for reproducing figure panels based on spatial transcriptomics analyses. To help navigate the wide variety of data and code, please refer to the following tables:

## Disclaimer

Reproducing these analyses with different/updated software versions or random generator seeds may result in slight variations. Please see the Methods section for details, and contact the corresponding authors if you have any questions regarding specific packages or data-preprocessing steps.
