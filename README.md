# Mattiuz2026_Manuscript_Code

This repository contains data and code relevant to spatial transcriptomics datasets presented in [Mattiuz R., et al. (In revision)](https://www.biorxiv.org/content/10.1101/2024.12.27.628014v1).

### Repository contents

This repository accompanies the raw + processed objects uploaded to GEO. 

### Downloading manuscript data

Raw human data and main processed objects containing annotations are available on NCBI GEO under accession number: [GSE270148](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE270148).
